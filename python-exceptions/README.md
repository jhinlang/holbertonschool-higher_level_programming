# Python Exceptions

This directory contains exercises and examples for handling Python exceptions.

## Topics Covered

- Exception basics
- Try, except, and finally blocks
- Raising exceptions
- Custom exceptions
- Exception handling best practices

## Files

- `*.py` - Python scripts demonstrating exception handling

## Usage

Run individual Python files:

```bash
python3 <filename>.py
```

## Learning Objectives

- Understand how to catch and handle exceptions
- Learn when to raise exceptions
- Create custom exception classes
- Write defensive code with proper error handling
